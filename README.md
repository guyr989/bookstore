# InsurTix Bookstore

A bookstore catalog manager. **ASP.NET Web API 2 (.NET Framework 4.8.1)** backend with an **XML-file database**, and an **Angular 17** frontend. Full CRUD by ISBN, HTML reports, Swagger, and per-environment (local/test/prod) XML data.
